Разработана программная модель учебной вычислительной машины Simple Computer, а также набор инструментов для её программирования.

В состав проекта входят:
Модель Simple Computer с эмуляцией:
- оперативной памяти (128 ячеек по 15 бит)
- процессора (АЛУ, управляющее устройство, регистры, кэш L1)
- подсистемы прерываний (сигналы, таймер)
- терминального ввода/вывода с псевдографикой

Библиотеки:
- mySimpleComputer — работа с памятью, флагами, командами
- myTerm — управление терминалом (цвета, курсор, очистка)
- myBigChars — вывод крупных символов и псевдографики
- myReadKey — неканонический ввод с клавиатуры

Трансляторы:
- Simple Assembler → бинарный код
- Simple Basic → код на ассемблере → бинарный код

Консоль управления моделью с интерактивным редактированием памяти, регистров, загрузкой/сохранением состояния и пошаговым выполнением.
