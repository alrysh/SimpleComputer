В состав проекта входит:
  Модель Simple Computer, которая эмулирует:
- оперативную память (128 ячеек по 15 бит)
- процессор (АЛУ, управляющее устройство, регистры, кэш L1)
- подсистему прерываний (сигналы, таймер)
- терминальный ввод/вывод с псевдографикой

  Библиотеки:
- mySimpleComputer — работа с памятью, флагами, командами
- myTerm — управление терминалом (цвета, курсор, очистка)
- myBigChars — вывод крупных символов и псевдографики
- myReadKey — неканонический ввод с клавиатуры

  Трансляторы:
- Simple Assembler → бинарный код
- Simple Basic → код на ассемблере → бинарный код

  Консоль управления SimpleComputer с интерактивным редактированием памяти, регистров, загрузкой/сохранением состояния и пошаговым выполнением.
